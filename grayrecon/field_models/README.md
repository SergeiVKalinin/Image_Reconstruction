# Field models

Grayscale reconstruction methods based on direct and latent stochastic fields.

This module will include:

- direct Gaussian-process reconstruction of the image;
- latent amplitude and phase models;
- periodic structural models;
- priors on wall position, shape, width, and phase discontinuity;
- posterior sampling and uncertainty estimation.

The observed grayscale image is generated from one or more latent fields. The internal structure of this module will be defined from the field reconstruction algorithms already developed.
