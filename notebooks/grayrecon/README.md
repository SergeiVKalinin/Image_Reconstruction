# GrayRecon notebooks

Executable notebooks that document and demonstrate the GrayRecon workflow.

The notebooks will cover:

1. generation of common synthetic grayscale reconstruction problems;
2. reconstruction using field models;
3. reconstruction using structural motif models;
4. reconstruction using quasi-MD dynamics;
5. comparison of the three reconstruction families on identical datasets.

The notebooks serve as the primary worked examples for the repository. Separate example scripts are not planned at this stage.
